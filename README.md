# 🌐 Internet Website

An interactive educational website that explains the Internet, network protocols, and how data travels between devices using visual animations.

## 📌 About The Project

This project provides a simple and interactive way to understand the Internet. It explains important networking concepts such as protocols, communication layers, and data transmission flow.

## ✨ Features

- 🌍 Overview of the Internet
- 🔗 Explanation of network protocols
- 📡 TCP/IP model visualization
- 🚀 Data transfer animation
- 💻 Client-server communication concept
- 📱 Responsive design
- 🎨 Interactive user interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📚 Concepts Covered

- Internet basics
- IP Address
- DNS
- HTTP / HTTPS
- TCP and UDP
- Network layers
- Data packet transmission
