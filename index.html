🌍 Overview of the Internet

The Internet is a global system of interconnected computer networks. It connects billions of devices using the TCP/IP protocol suite and allows users to communicate, browse websites, transfer files, send emails, stream videos, and much more.


Major Components

💻 End Systems

Computers, mobiles, servers and IoT devices.

📡 Communication Links

Fiber optic, Wi-Fi, satellite and mobile networks.

🌍 Routers

Forward packets between different networks.

🏢 ISP

Provides Internet access to homes and businesses.

📚 Protocol Layering

Protocol layering divides networking into logical layers. Each layer performs a specific task and communicates with the layer directly above and below it.

Application Layer

Provides services like HTTP, FTP, DNS and Email.

Transport Layer

Uses TCP and UDP for end-to-end communication.

Internet Layer

Uses IP for routing packets between networks.

Network Access Layer

Transfers frames over Ethernet and Wi-Fi.

🌐 Application Layer

The Application Layer is the top layer of the TCP/IP model. It provides network services directly to users and applications. When you browse websites, send emails, transfer files, or perform DNS lookups, you are using application layer protocols.


Main Responsibilities

  • ✔ User communication
  • ✔ Data exchange between applications
  • ✔ Web browsing
  • ✔ Email communication
  • ✔ File transfer
  • ✔ Name resolution

📂 Examples of P2P Networks

🧲 BitTorrent

Users download pieces of files from multiple peers.

⛓ Blockchain

Multiple computers maintain a distributed ledger.

🎮 Online Gaming

Players can communicate directly in distributed systems.

✅ Advantages of P2P

  • High scalability
  • Resource sharing
  • No single point of failure
  • Lower infrastructure cost
  • Efficient file distribution

⚠ Disadvantages of P2P

  • Security problems
  • Difficult management
  • Data privacy issues
  • Variable performance
